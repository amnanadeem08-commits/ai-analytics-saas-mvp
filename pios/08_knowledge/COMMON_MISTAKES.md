# Common Mistakes

1. Inventing features not present in code or labeling them as shipped
2. Skipping PIOS status/sprint read and re-deriving architecture from memory
3. Editing Streamlit pages with heavy logic instead of extracting services
4. Adding optional flags that change default behavior (breaks backward compatibility)
5. Marking S3/K8s/SSO/billing gateway as done when they are roadmap/KI items
6. Deleting or weakening tests to pass gates
7. Leaving PROJECT_STATUS stale after a task
