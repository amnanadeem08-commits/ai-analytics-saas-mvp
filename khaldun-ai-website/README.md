# Khaldun AI Website

Marketing site for **Khaldun AI** — an AI-powered data analytics company.

## Stack

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- Framer Motion
- Static export (`output: "export"`)

## Develop

```bash
npm install
npm run dev
```

## Build static site

```bash
npm run build
```

Output lands in `out/`.

## Content

Edit typed content in `content/site.ts`.
