# Typography Guide

Defined in `frontend/design_system/typography.py`.

| Role | Size | Weight | Usage |
|------|------|--------|-------|
| Display | 2rem | 800 | Rare hero titles |
| Heading | 1.35rem | 800 | Page titles (`page_header`) |
| Subheading | 1.1rem | 700 | Section titles |
| Body | 0.95rem | 400 | Paragraphs |
| Caption | 0.8rem | 500 | Hints, secondary labels |
| Code | 0.85rem | 500 | Inline code / IDs |
| Metric | 1.5rem | 800 | KPI values |

**Fonts:** Segoe UI / Source Sans 3 stack for UI; Cascadia Code / Source Code Pro for mono.

CSS classes: `.ds-display`, `.ds-heading`, `.ds-subheading`, `.ds-body`, `.ds-caption`, `.ds-code`, `.ds-metric-value`.
