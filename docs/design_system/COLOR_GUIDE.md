# Color Guide — Khaldun AI DataBot

Semantic color tokens live in `frontend/design_system/colors.py`.

| Token | Hex | Use |
|-------|-----|-----|
| Primary | `#174A7C` | Brand actions, links, active nav, metric values |
| Secondary | `#243B53` | Supporting chrome, archived status |
| Accent | `#B88322` | Highlights, secondary series |
| Success | `#15803D` | Completed, ready, positive |
| Warning | `#B45309` | Caution, cancelled |
| Danger | `#B91C1C` | Errors, failed, destructive |
| Info | `#0369A1` | Running / informational |
| Background | `#F5F7FA` | Page canvas / plot background |
| Surface | `#FFFFFF` | Cards, panels |
| Border | `#CBD5E1` | Dividers, card edges |
| Muted | `#64748B` | Captions, inactive steps |
| Text | `#0F172A` | Body copy |

CSS variables: `--ds-primary`, `--ds-success`, … also mirrored to existing `--brand-primary` for compatibility.

Chart palette: see [Chart Guide](./CHART_GUIDE.md).
